# FPD Style Layer

## Overview
The **Style Layer** defines the visual design language of the application. It ensures consistency across all components.

## Responsibilities
- **Design Tokens**: Colors, spacing, radii, shadows.
- **Typography**: Font families, sizes, weights, line heights.
- **Themes**: Light/Dark mode definitions.

## Rules
- **Visuals Only**: No logic or behavior.
- **Universal**: Used by the Component and Navigation layers (for screens).
