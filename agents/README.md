# Agents Configuration

This directory contains configuration files and guidelines for AI agents working with Flutter projects.

## Purpose

The agents configuration provides:
- Base setup and configuration for AI development assistants
- Guidelines for agent behavior and decision-making
- Standards for code generation and project interaction

## Usage

These configurations are designed to help AI agents:
- Understand project structure and conventions
- Follow established coding standards
- Make appropriate development decisions
- Maintain consistency across the project
